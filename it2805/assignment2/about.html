Questions

  1. The server hosts files and serves the client via some program which the client can send requests to.
  2. The client communicates with the server and can request certain info such as a webpage. When it is received it will render it locally.
  3. An IP address is an identificator in the network layer for devices that communicate over a network. Think telephone number for network devices.
  4. DNS allows us to resolve domain names such as vg.no to the correct IP address so we do not have to remember and need to be locked to static IP addresses. Think telephone address book for devices.
  5. The Internet is the interconnected set of networks that most people are connected to which hosts many different applications and services. WWW is part of the Internet and contains HTML webpages.
  6. HTML is a markup language with a bunch of markup tags for creating webpages. The tags are used to mark different parts of the page such as paragraphs, headers and images.
